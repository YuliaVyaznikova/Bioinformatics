# Изучение представленности гена в таксонах

## 1. Выбран ген CNGA3
- Cyclic nucleotide-gated channel alpha 3 (кодирует альфа-субъединицу циклонуклеотид-зависимого ионного канала колбочек сетчатки, критически важного для цветового зрения). Мутации в гене CNGA3 вызывают ахроматопсию (ACHM2).

## 2. Параметры BLAST, использованные для поиска гомологичных генов
Для поиска гомологичных генов использовался алгоритм NCBI BLAST со следующими параметрами:
- Program: blastn
- Database: refseq_rna (Reference RNA sequences)
- Query: Homo sapiens CNGA3 transcript variant 1, mRNA (NM_001298.3)
- Algorithm: Somewhat similar sequences (blastn) - позволяет найти более далёких родственников за пределами млекопитающих
- Expect threshold: 1e-20
- Megablast: false
- Hitlist size: 100
- Word size: default (11)
- Gap costs: default
- Filtering: default (low complexity masking)

Выравнивание выполнено с помощью Clustal Omega (EBI). Включает 11 последовательностей, длина выравнивания: 7150 нуклеотидов.

## 5. Краткий анализ консервативности

В полученном множественном выравнивании наблюдается умеренная степень консервативности, что характерно для нуклеотидных последовательностей генов, кодирующих трансмембранные ионные каналы.

- Средний уровень консервативности (доля преобладающего нуклеотида без учёта gap): 0.742
- Полностью консервативных колонок (100% идентичность): 2650 из 7150 (37.1%)
- Высококонсервативных колонок (≥90%): 2801 (39.2%)
- Среднеконсервативных колонок (≥70%): 825 (11.5%)
- Низкоконсервативных колонок (<70%): 3524 (49.3%)

Протяжённые высококонсервативные блоки (≥10 подряд идущих полностью консервативных нуклеотидов):
- Позиции 1–112 (длина 112 нт) - начало выравнивания, соответствует 5'-концу транскрипта
- Позиции 140–164 (длина 25 нт)
- Позиции 166–181 (длина 16 нт)
- Позиции 338–358 (длина 21 нт)
- Позиции 5601–5622 (длина 22 нт)
- Позиции 5713–5740 (длина 28 нт)
- Позиции 5845–5862 (длина 18 нт)
- Позиции 6123–6134 (длина 12 нт)
- Позиции 6401–6425 (длина 25 нт)
- Позиции 6436–7150 (длина 715 нт) - 3'-конец транскрипта

Наиболее консервативные участки соответствуют кодирующим областям трансмембранных доменов (S1–S6) и поры ионного канала, а также домену связывания циклических нуклеотидов (cNMP-binding domain) на С-конце. Наибольшая вариабельность наблюдается в нетранслируемых областях (5'-UTR и 3'-UTR) и в N-концевом цитоплазматическом домене, которые менее ограничены функциональными требованиями.

Примечание: у рыб (Danio rerio, Oryzias latipes, Gadus morhua) ген представлен паралогом cnga3a - результатом третьего раунда полногеномной дупликации у лучеперых рыб (teleost-specific genome duplication).

## 6. Объединяющий таксон
- Латинское название: Vertebrata
- Русское название: Позвоночные
- Краткое описание: Vertebrata - подтип хордовых животных, включающий млекопитающих, птиц, рептилий, земноводных и рыб. Для представителей группы характерно наличие позвоночного столба (позвонков), черепа, развитой центральной нервной системы (головного и спинного мозга) и сложных органов чувств, включая глаза камерного типа с сетчаткой, в которой экспрессируется ген CNGA3. Все 11 проанализированных видов относятся к подтипу позвоночных; это ближайший таксон, объединяющий все найденные гомологи гена CNGA3.
